# WP Last Login Tracker

A powerful WordPress plugin that helps administrators monitor user activity by tracking login dates, login IP addresses, inactive users, and exporting activity reports.

## 🚀 Overview

WP Last Login Tracker provides valuable insights into user engagement and account activity. It records login information automatically and presents it through an intuitive admin interface, helping site owners manage users more effectively.

Whether you're running a membership site, e-learning platform, community website, intranet, or customer portal, this plugin gives you the visibility needed to understand user activity.

## ✨ Features

### User Login Tracking

* Track each user's last login date and time
* Record login IP addresses
* Display login information on user profile pages
* View recent login information

### User Management

* Add Last Login column to the Users table
* Add Login IP column to the Users table
* Sort and filter users based on login activity
* Quickly identify inactive accounts

### Inactive User Detection

* View users who have not logged in for a specified period
* Improve account management and identify inactive users
* Dedicated Inactive Users administration page

### Reporting & Export

* Export user activity data to CSV
* Export login information for administrative review
* Easy access to login records

### Dashboard Insights

* Dashboard widget showing recent login activity
* Quick overview of active users
* Convenient administrative monitoring

### Developer Friendly

* REST API support
* Translation ready
* WordPress coding standards compatible
* Extensible architecture

## 📦 Installation

### From GitHub

1. Download the latest release.

2. Upload the plugin folder to:

   wp-content/plugins/

3. Activate the plugin from the WordPress Admin Dashboard.

4. Configure plugin settings if required.

### Manual Installation

1. Download the plugin ZIP file.
2. Navigate to **Plugins → Add New → Upload Plugin**.
3. Upload the ZIP file.
4. Activate the plugin.
